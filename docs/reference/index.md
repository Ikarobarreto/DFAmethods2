  : rho Detrended Cross-Correlation Coefficient

  : rho Detrended Multiple-Correlation Coefficient

- [`plotdcca()`](https://ikarobarreto.github.io/DFAmethods2/reference/plotdcca.md)
  : Plot DCCA

## Detrended fractal regression

- [`betadfa()`](https://ikarobarreto.github.io/DFAmethods2/reference/betadfa.md)
  : Beta DFA
- [`sbdfa()`](https://ikarobarreto.github.io/DFAmethods2/reference/sbdfa.md)
